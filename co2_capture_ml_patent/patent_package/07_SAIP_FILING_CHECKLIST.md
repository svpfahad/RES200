# Saudi Filing Checklist

Status: attorney-review draft, not legal advice. Saudi filing should be handled by a Saudi patent agent/attorney.

## Saudi Patentability Frame

Frame the invention as a computer-implemented industrial process for screening CO2 capture solvents, not as a mathematical method. Saudi patent law provides for patents on inventions that are new, involve an innovative step, and are industrially applicable. The invention may be a product, industrial process, or related to either. Saudi law excludes discoveries, scientific theories, mathematical methods, pure mental/business rules, and certain biological/medical method categories.

## Required Filing Materials For Attorney

- patent application form;
- applicant name, capacity, address, place of residence/work;
- inventor name and address;
- agent name and address if applicable;
- priority and disclosure information;
- specification;
- claims;
- abstract;
- drawings if needed;
- assignment document if applicant is not the inventor;
- power of attorney for Saudi agent;
- filing fee;
- priority document and translation if claiming priority;
- Arabic translation plan.

## Specification Format

Saudi implementing regulations state that the specification includes, in order:

1. Abstract;
2. Full Description;
3. Claims;
4. Drawings.

The full description should include:

1. Background of the Invention;
2. General Description of the Invention;
3. Brief Explanation of the Drawings;
4. Detailed Description.

For this invention, the Saudi draft should emphasize:

- technical field: CO2 capture solvent screening;
- technical problem: unsupported ML recommendations for new chemistry;
- industrial use: carbon capture process development and solvent testing;
- technical effect: calibrated abstention and reason-coded experimental triage;
- best implementation: local Python workflow and evidence suite;
- examples: public-data results and candidate decisions.

## Arabic / Translation Notes

The Saudi package will need Arabic handling. Scientific terms should be used consistently. Chemical names, formulas, SMILES, and IUPAC terms may require careful bilingual treatment. Counsel should decide whether to file Arabic first or file with a permitted translation path.

## Non-Resident Applicant

If the applicant resides outside Saudi Arabia, a Saudi authorized agent is required. A power of attorney may need approval/certification depending on applicant location and counsel practice.

## Disclosure And Priority

Before filing, collect all disclosure history:

- publications;
- preprints;
- conference abstracts;
- posters;
- thesis deposits;
- GitHub or public code;
- demos;
- investor/customer decks;
- grant applications that became public;
- emails to non-confidential recipients.

Saudi prior art is broad and public disclosure can be damaging. Do not publish, present, or send enabling details outside confidentiality before counsel confirms the filing sequence.

## Filing Route Options

1. U.S. provisional first, then PCT or Saudi/foreign filings within priority deadlines.
2. Direct Saudi filing if Saudi rights are urgent.
3. PCT route designating Saudi Arabia, then Saudi national phase.

Counsel should decide the route after reviewing budget, target markets, public disclosure risk, ownership, and whether rapid Saudi filing is commercially important.
