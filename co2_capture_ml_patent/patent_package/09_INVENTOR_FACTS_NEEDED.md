# Inventor Facts Needed Before Filing

Status: required intake checklist.

## Inventor And Ownership

- Legal name of each inventor.
- Citizenship/residence of each inventor.
- Mailing address of each inventor.
- Who contributed to the inventive concept, not just coding or routine testing.
- Employer, university, or sponsor at the time of invention.
- Employment agreements, IP assignment agreements, consulting agreements, contractor agreements.
- Any government, university, or grant funding.
- Whether any collaborator contributed data, code, model design, candidate chemistry, or experimental design.

## Disclosure History

For each disclosure, provide date, recipient/audience, confidentiality status, and copy:

- papers or preprints;
- posters or talks;
- thesis submissions;
- GitHub/public repositories;
- social media or blog posts;
- grant applications or public abstracts;
- investor decks;
- customer demos;
- emails to non-confidential recipients;
- uploaded datasets or model files.

## Technical Facts

- Exact first date the core idea was conceived.
- Exact first date a working implementation existed.
- Who wrote each major part of the code.
- Which datasets were used.
- Which candidate solvents are confidential.
- Any wet-lab experiments already performed.
- Any planned experiments.
- Whether the system has been used to select real candidates.
- Whether any output has commercial or confidential value.

## Filing Decisions Needed

- Applicant name: individual, company, university, or joint.
- First filing jurisdiction.
- Budget for U.S. provisional, PCT, Saudi national, and later national phases.
- Whether Saudi Arabia is a primary market.
- Whether the code should be published, kept confidential, or partially disclosed.
- Whether candidate solvent compositions should be patented separately after lab validation.
